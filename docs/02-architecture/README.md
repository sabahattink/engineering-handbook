# 02 — Architecture

Cross-cutting architectural principles that apply regardless of language,
framework, or platform.

This chapter is scoped deliberately narrowly: it does **not** contain
framework-specific guidance (e.g. "how to structure a Next.js app" or
"NestJS module layout"). That kind of guidance belongs in the repository it
applies to, or in that ecosystem's own conventions — referencing, not
duplicating, the principles defined here.
